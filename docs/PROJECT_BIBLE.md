# Mission

ZeroVault is an offline-first desktop application designed to securely encrypt and decrypt files and folders while ensuring that user data never leaves the local machine.

The goal is to provide strong, modern encryption through an interface that is simple enough for everyday users and trustworthy enough for professionals.

---

# Vision

Build a professional-grade encryption application that users can trust with their most valuable data.

ZeroVault prioritizes security, transparency, performance, maintainability, and user experience over rapid feature development.

---

# Core Principles

1. Security before convenience.
2. Offline by default.
3. Never invent cryptography.
4. Never store user passwords.
5. Encryption keys exist only in memory.
6. Every dependency must have a purpose.
7. Every commit must compile successfully.
8. AI-generated code must always be reviewed.
9. Documentation evolves together with the code.
10. Simplicity is preferred over unnecessary complexity.
